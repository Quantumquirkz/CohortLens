# Estado Actual de la Aplicación Móvil

## Arquitectura
- **Framework**: Expo (React Native)
- **Lenguaje**: TypeScript
- **Navegación**: React Navigation
- **Gestión de Estado**: React Context API
- **Autenticación**: JWT

## Estructura de Archivos
- **Pantallas**: `/app`
