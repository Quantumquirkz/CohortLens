# Contributor guide

See the repository **[CONTRIBUTING.md](https://github.com/Quantumquirkz/CohortLens/blob/main/CONTRIBUTING.md)** for branches, commits, and PR expectations.

## Quick setup

- **Contracts**: install [Foundry](https://book.getfoundry.sh); `forge test` in `packages/contracts`.
- **Backend**: Python 3.11+, `pip install -r requirements.txt`, `uvicorn app.main:app`.
- **Frontend**: Node 18+, `npm install`, `npm run dev`.
- **Docs**: `cd docs && npm install && npm start`.

## Code of conduct

Contributors must follow the **[Code of Conduct](https://github.com/Quantumquirkz/CohortLens/blob/main/CODE_OF_CONDUCT.md)**.
