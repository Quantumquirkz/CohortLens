# Introduction

**CohortLens** is a monorepo for cohort analysis and on-chain machine learning:

- **Smart contracts** (`packages/contracts`) — Foundry, registry, oracle, tokenomics.
- **Backend** (`packages/backend-ai`) — FastAPI, ML, subgraph GraphQL, web3.py.
- **Frontend** (`packages/frontend`) — Next.js, Wagmi, React Query.
- **Indexers** (`packages/indexers`) — The Graph subgraphs per protocol and chain.

Use the sidebar for architecture, deployment, API reference, and security practices.
