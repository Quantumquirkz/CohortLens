# Raw Data Directory

Place your `Customers.csv` file here before running the pipeline.

## Dataset

Download the Customers dataset from Kaggle:
- [Customers Dataset](https://www.kaggle.com/datasets/datascientistanna/customers-dataset)

Required columns: `CustomerID`, `Gender`, `Age`, `Annual Income ($)`, `Spending Score (1-100)`, `Profession`, `Work Experience`, `Family Size`.
